# WIFI.ID-auto-login

Script untuk selalu konek ke WIFI.ID hotspot, untuk yang ingin **memanfaatkan** 
bandwidth WIFI.ID :).
Di eksekusi melalui cron job. 

## Feature ##

 - Multi account
 - Mikroik support

## Instalasi ##
  
        clone repo
        cd <to diretory>
        python setup.py install
        copy config ke /etc
	tambahkan cron job i.e */1 * * * * auto_login_wifi.id.py
